# UPX 脱壳结果总结

## 结论：✅ 脱壳成功

扫雷程序 `saolei.exe` 的 UPX 加壳版本 `ke_saolei.exe` 已成功脱壳，脱壳后程序可正常运行。

---

## 脱壳三步走

### 1️⃣ 找 OEP → ESP 定律
- x64dbg 载入加壳程序
- `pushad` 后对 ESP 指向地址下硬件访问断点
- F9 运行，断在 `popad`，单步到达 **OEP: 0x01003E21**

### 2️⃣ Dump + 修 IAT → Scylla
- 在 OEP 处 Dump 内存
- IAT Autosearch → Get Imports → Fix Dump

### 3️⃣ 区段名还原 → CFF Explorer
- `UPX0` → `.text`
- `UPX1` → `.rdata`
